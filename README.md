# Homebrew-Amiga

A repository for **Amiga Development** related brews.


## Requirements

* [Homebrew](https://github.com/mxcl/homebrew)


## Installation

Tap the repository:

	brew tap tditlu/amiga

Then install wanted formula:

	`brew install vbcc`

	`brew install vasm`

	`brew install vlink`

	`brew install shrinkler`
